### 3️⃣ `configs/ufw-rules.txt`

```text
# Default policy
ufw default deny incoming
ufw default allow outgoing

# SSH
ufw allow 22/tcp

# Samba
ufw allow 137,138/udp
ufw allow 139,445/tcp

# NFS
ufw allow from 192.168.1.0/24 to any port 2049
ufw allow from 10.70.133.0/24 to any port 2049

# mDNS (optional)
ufw allow 5353/udp

# Tailscale (only when enabled)
ufw allow in on tailscale0
```

**This file exists so you never “experiment” with UFW again.**
